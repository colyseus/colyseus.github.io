Did you know that making multiplayer games is not as hard as it used to be?

Rules

1. The source-code of the game should be available publicly (hosted on GitHub, BitBucket, GitLab, etc.)

2. The game should be playable with 2 or more online players

3. Make it easy for people to test your game by providing a playable link or download build


FAQ

How the theme will be decided? It will be the same theme from Ludum Dare 46 Game Jam, which will be happening at the same time. You can submit your game on both game jams.

How to submit your game: A link to submit your game will be available after April 17th.

Can I use an ongoing project to participate in the Jam? No, but you can re-use existing pieces

What's the prize? Well, you will (hopefully) have a playable multiplayer game by the end of the jam, that's a great prize!
