Sponsors

A big thanks to the companies supporting the development of Colyseus.

If you're a solo indie developer, small or large game studio, Colyseus will allow you to deliver realtime or turn-based multiplayer games in record time.

Features

  • Matchmaking

    Match players into game rooms, created on-the-fly.

  • State synchronization

    Mutate the game state in the server, and have all the clients automatically up-to-date.

  • Message-passing

    Listen for and send messages on both client-side and server-side.

  • Authoritative

    Colyseus encourages an authoritative approach to prevent hacked clients.
